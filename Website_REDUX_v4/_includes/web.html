iWork.

Web.

I built this website using HTML5, CSS3, and Javascript. I started to build websites around the fourth grade (circa 2001). I was only using HTML at the time and wasn't interested learning any CSS or Javascript (and curiosity for what was beyond) didn't happen until high school. The site is hosted through GitHub pages, which exclusively uses an engine called Jekyll (created by Tom Preston-Werner of Github) to generate a blog from static pages.

Optimizing my website under Jekyll gave myself a more intuitive and relaxing web-designing experience. I'm quite thankful to have spent the time porting my old site to this Ruby-based engine. I would also like to thank Jarrell Waggoner for helping me understand Jekyll as well as informing me of its' existence. Firefox is my favorite web browser and BBEdit is my text editor!
